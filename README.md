# Sublime Tests File  
> 快速 添加/移除 一个文件夹到当前窗口, 文件夹中包含各种后缀的test文件, 方便调试.  


## 安装  
1. 进入Sublime的Packages目录.  
2. `git clone https://github.com/floydawong/sublime-tests-file.git`  


## 使用方法  
1. <kbd>ctrl + shift + p</kbd> 打开命令行面板.  
2. 输入 `Tests: `.  
3. 选择并回车.  


## 如何修改后缀名  
打开工程中的`suffix.txt`文件, 修改并保存. `reload`此插件即可.
